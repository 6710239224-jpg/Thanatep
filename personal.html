ประวัติการศึกษา

  • ระดับมัธยมศึกษาตอนต้น – พระยามนธาตุราชศรีพิจิตร์
  • ปัจจุบันกำลังศึกษาในระดับ ปวช.2
  • สาขา คอมพิวเตอร์ธุรกิจ (Global Program)
  • วิทยาลัยอาชีวศึกษา

ความสามารถและทักษะ

  • พื้นฐานการเขียนเว็บไซต์ด้วย HTML และ CSS
  • พื้นฐานภาษาโปรแกรม JavaScript และ Python
  • ใช้งาน Microsoft Word, Excel, PowerPoint
  • ค้นคว้าข้อมูลผ่านอินเทอร์เน็ตอย่างมีประสิทธิภาพ
  • สามารถสื่อสารภาษาอังกฤษพื้นฐาน

ประสบการณ์ / กิจกรรม

  • ฝึกเขียนเว็บไซต์เพื่อส่งงานในรายวิชาคอมพิวเตอร์
  • ทำรายงานและนำเสนอผลงานหน้าชั้นเรียน
  • ทำงานกลุ่มและฝึกการทำงานเป็นทีม

ความสนใจ

  • การเขียนโปรแกรมและพัฒนาเว็บไซต์
  • เทคโนโลยีและนวัตกรรมด้านไอที
  • การออกแบบเว็บไซต์

เป้าหมายในอนาคต
